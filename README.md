# 📞 Click-to-Call v2

AI 음성 기반 호출 기능이 내장된 긴급 연락 시스템입니다.  
사용자가 설정한 음성 명령 또는 버튼 인터페이스를 통해 즉시 전화 연결이 가능합니다.

---

## 📂 폴더 구성

| 파일명 | 설명 |
|--------|------|
| `.env` | 민감 정보 환경 설정 (Twilio 등) |
| `Dockerfile` | Docker 기반 실행 설정 파일 |
| `index.js` | 메인 서버 로직 |
| `package.json` | Node.js 패키지 설정 |

---

## 🚀 실행 방법

### 로컬 실행

```bash
npm install
